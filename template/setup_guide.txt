An ios build is triggered by setting the tag ios or ios-dev.
The worfklow setups the capacitor project and pushes the result to ios or ios-dev branch.
The ios build and signing itself is executed on xcode cloud.

Now you must configure xcode cloud to build on push.
The first time this setup has to be done by opening the project once in xcode.
Then xcode will ask you to link your repo.
